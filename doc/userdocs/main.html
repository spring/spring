What is Spring?

Spring is a project aiming to create a new Real Time Strategy (RTS) engine.

The Spring engine has many games that have been made for it. Some of these games are modifications of the commerical game Total Annihilation (e.g. Balanced Annihilation or XTA), others are completely original games (e.g. Gundam, Kernel Panic, Evolution, Spring: 1944, and Simbase).
